  🎓 COMO ADICIONAR NOVAS FUNCIONALIDADES
═════════════════════════════════════════════════════════════════

Para adicionar uma nova página:
  1. Crie arquivo em: src/pages/NovaPage.jsx
  2. Importe em: src/App.jsx
  3. Adicione rota em: <Routes>

Para adicionar novo serviço:
  1. Crie arquivo em: src/services/novoService.js
  2. Exporte funções
  3. Importe e use nos componentes

Para customizar estilos:
  Busque pelas cores principais:
  • #FF8C00  (Laranja - primária)
  • #F4A460  (Laranja clara - secundária)
  • #00BFFF  (Azul - destaque)

  🔗 ENDPOINTS ESPERADOS DA API
═════════════════════════════════════════════════════════════════

POST   /login                    → Autenticação
POST   /logout                   → Logout
POST   /recuperar-senha          → Recuperação
POST   /redefinir-senha          → Redefinição

GET    /produtos                 → Listar todos
POST   /produtos                 → Criar
PUT    /produtos/{id}            → Atualizar
DELETE /produtos/{id}            → Deletar

GET    /fornecedores             → Listar todos
POST   /fornecedores             → Criar
PUT    /fornecedores/{id}        → Atualizar
DELETE /fornecedores/{id}        → Deletar
